# WEEK 8 LESSON PLAN

## 1. Lesson Information

| Field | Value |
|-------|--------|
| **Week** | 8 |
| **Topic** | Flexbox: Layout with Flex |
| **Duration** | 2–3 Hours |

---

## 2. Learning Objectives

By the end of this lesson, students will be able to:

- Build a horizontal nav (e.g. logo left, links right) with Flexbox.
- Build a row of cards that wrap on smaller screens.
- Use flex-wrap and something like flex: 1 1 300px on each card; optional: simple hero centered with Flexbox.

---

## 3. Prior Knowledge

Students should understand:

- Responsive design, media queries, units.
- Card layout and basic CSS.

---

## 4. Materials Needed

- Computer, code editor, internet access
- Previous project (cards or 2-page site)

---

## 5. Key Concepts

- **Flexbox:** display: flex on the container.
- Main vs cross axis.
- **Container:** flex-direction, justify-content, align-items, gap, flex-wrap.
- **Items:** flex (e.g. flex: 1 1 300px).
- Centering; simple nav + card grid.

---

## 6. Lesson Procedure

### A. Common Mistakes Review (5–10 minutes)

Use [Common-Mistakes-And-Debugging.md](../notes/Week-1/Common-Mistakes-And-Debugging.md) — **Session 6**. Flex direction, wrap, flex shorthand.

### B. Introduction of Core Concept (40–60 minutes)

Instructor explains: display: flex; main/cross axis; justify-content, align-items, gap, flex-wrap; flex on items (flex: 1 1 300px); live coding: nav then row of cards that wrap.

### C. Guided Practice (30–45 minutes)

Students: build Flexbox nav (logo left, links right); build row of 3 cards with flex-wrap and flex on each card.

### D. Independent Practice (30–60 minutes)

**Main exercise:** One page with: Flexbox nav and a row of 3 cards that wrap. Use flex-wrap and flex: 1 1 300px (or similar) on each card. Optional: simple hero (title + text) centered with Flexbox.

**Real-world:** “Same layout pattern as many portfolios and marketing pages.”

**Milestone:** Responsive Layout Developer.

### E. Assessment

Nav and card row work; layout responds to screen size; no console errors.

### F. Wrap-Up Discussion (10–15 minutes)

What was difficult? What did you debug? How is this used in real life?

### G. Homework (Light / Optional)

Read “What is JavaScript?” and “Variables and types” in Week 4 material. Add `<script src="script.js"></script>` before `</body>` and console.log("Hello"); in script.js.
