# WEEK 5 LESSON PLAN

## 1. Lesson Information

| Field | Value |
|-------|--------|
| **Week** | 5 |
| **Topic** | What CSS Is and Basic Styling |
| **Duration** | 2–3 Hours |

---

## 2. Learning Objectives

By the end of this lesson, students will be able to:

- Link a CSS file and apply rules to body, headings, and paragraphs.
- Change text colour, size, and alignment without touching HTML structure.
- Style the 2-page site: one font, background colour, styles for h1/h2/p, links and :hover, center content with max-width and margin: 0 auto.

---

## 3. Prior Knowledge

Students should understand:

- Multi-page HTML (index.html, contact.html) with semantic structure.
- Folder structure and file naming.

---

## 4. Materials Needed

- Computer, code editor, internet access
- Week 4 two-page site

---

## 5. Key Concepts

- **What CSS is:** structure (HTML) vs presentation (CSS).
- Linking external stylesheet: `<link rel="stylesheet" href="styles.css">`.
- **Syntax:** selector `{ property: value; }`.
- **Basic styling:** color, background-color, font-size, font-family, text-align.
- Links: color, text-decoration; :hover.

---

## 6. Lesson Procedure

### A. Common Mistakes Review (5–10 minutes)

Use [Common-Mistakes-And-Debugging.md](../notes/Week-1/Common-Mistakes-And-Debugging.md) — **Session 3** (first CSS). Wrong path in link, selector typo, hard refresh.

### B. Introduction of Core Concept (40–60 minutes)

Instructor explains: structure vs presentation; linking stylesheet; selector and property syntax; color, font-size, font-family, text-align; styling links and :hover; centering with max-width and margin: 0 auto. Live coding on the 2-page site.

### C. Guided Practice (30–45 minutes)

Students: add link to styles.css; create styles.css; style body (font, background); style h1, h2, p; style links and :hover; center main with max-width and margin: 0 auto.

### D. Independent Practice (30–60 minutes)

**Main exercise:** Style the 2-page site from Week 4. Must include: one font, one background colour; different styles for h1, h2, p; styled links (e.g. colour, no underline) and :hover; main content centered with max-width and margin: 0 auto.

**Real-world:** “You’re defining the look of a real site: typography and colours.”

### E. Assessment

Working styles on both pages; no console errors; CSS loads (check Network/Validator).

### F. Wrap-Up Discussion (10–15 minutes)

What was difficult? What did you debug? How is this used in real life?

### G. Homework (Light / Optional)

Read about the box model (padding, margin, border) and classes/IDs.
