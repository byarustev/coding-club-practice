# WEEK 2 LESSON PLAN

## 1. Lesson Information

| Field | Value |
|-------|--------|
| **Week** | 2 |
| **Topic** | Essential HTML Tags: Headings, Paragraphs, Links, Images, Lists |
| **Duration** | 2–3 Hours |

---

## 2. Learning Objectives

By the end of this lesson, students will be able to:

- Choose the right tag for headings, text, links, images, and lists.
- Use `alt` on images and correct paths in `src` and `href`.
- Build a single-page “About me” or “My favourite thing” with name as `h1`, paragraphs, one image (with alt), one list, and at least one external link and one `mailto:` link.

---

## 3. Prior Knowledge

Students should understand:

- Valid HTML document structure (DOCTYPE, html, head, body).
- Basic folder structure and file naming (`index.html`).

---

## 4. Materials Needed

- Computer
- Code editor
- Internet access
- Previous project files (Week 1)

---

## 5. Key Concepts

- **Headings:** `h1`–`h6`; one `h1` per page.
- **Paragraphs:** `p`.
- **Links:** `a` with `href` (external, `mailto:`, anchor).
- **Images:** `img` with `src` and `alt`.
- **Lists:** `ul`/`ol`/`li`.
- **Attributes:** `id`, `class`, `src`, `href`, `alt`. Closing tags; indentation.

---

## 6. Lesson Procedure

### A. Common Mistakes Review (5–10 minutes)

- Use [Common-Mistakes-And-Debugging.md](../notes/Week-1/Common-Mistakes-And-Debugging.md) — **Session 2**.
- Fix 1–2 common errors together (e.g. wrong path for links, missing `alt` on images).

### B. Introduction of Core Concept (40–60 minutes)

Instructor explains:

- Headings hierarchy and one `h1` per page.
- Paragraphs and line breaks.
- Links: external, `mailto:`, anchor links.
- Images: `src`, `alt`, relative paths.
- Unordered and ordered lists.
- Live coding demonstration of a short “About me” snippet.

### C. Guided Practice (30–45 minutes)

Students follow instructor to build:

- One `h1` (name or title).
- Two or three `p` elements.
- One image with correct `alt`.
- One list (e.g. hobbies or steps).
- One external link and one `mailto:` link.

### D. Independent Practice (30–60 minutes)

**Main Weekly Exercise:**

Build a single-page “About me” or “My favourite thing” that must include:

- Name as `h1`.
- Short paragraphs.
- One image (with alt).
- One list (hobbies or steps).
- At least one external link and one `mailto:` link.

**Real-world:** “Same building blocks as any profile or product page.”

### E. Assessment

Students must demonstrate:

- Correct use of headings, paragraphs, links, images, lists.
- Working code without validation errors.
- All links and image paths working.

### F. Wrap-Up Discussion (10–15 minutes)

- What was difficult?
- What did you debug?
- How is this used in real life?

### G. Homework (Light / Optional)

- Read about semantic HTML (header, nav, main, section, footer). No coding homework if time is tight.
