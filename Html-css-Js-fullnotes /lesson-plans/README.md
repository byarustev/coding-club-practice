# HTML, CSS & JavaScript — School Programme Lesson Plans

**Duration:** 18 Weeks  
**Contact Time:** 2–3 Hours Per Week  
**Approach:** 1–2 Core Concepts Per Week

---

## Programme Overview

This programme teaches web development fundamentals at a school-friendly pace. Each week focuses on **one or two core concepts** with one main practical exercise, common-mistakes review, and light homework.

| Part | Weeks | Focus |
|------|--------|--------|
| **Part 1: HTML** | 1–4 | Document structure, essential tags, semantic HTML, multi-page sites |
| **Part 2: CSS** | 5–8 | Styling, box model, responsive design, Flexbox |
| **Part 3: JavaScript** | 9–15 | Variables, functions, DOM, events, arrays, localStorage |
| **Part 4: Integration** | 16–18 | Project structure, Git (optional), Fetch (optional), final project |

---
