# WEEK 4 LESSON PLAN

## 1. Lesson Information

| Field | Value |
|-------|--------|
| **Week** | 4 |
| **Topic** | Multi-Page Sites and Linking |
| **Duration** | 2–3 Hours |

---

## 2. Learning Objectives

By the end of this lesson, students will be able to:

- Create a second HTML page and link from the first.
- Copy the same nav into both pages and fix links so they work from either page.
- Use a clear folder structure (e.g. `index.html`, `contact.html`, `images/`).

---

## 3. Prior Knowledge

Students should understand:

- Semantic HTML (header, nav, main, section, footer).
- Relative paths for links and images.

---

## 4. Materials Needed

- Computer
- Code editor
- Internet access
- Week 3 project (semantic single page)

---

## 5. Key Concepts

- Multi-page sites: second file (e.g. `contact.html`).
- Same nav on each page.
- Relative links between pages (`index.html`, `contact.html`).
- Folder structure: `index.html`, `contact.html`, `images/` (if used).

---

## 6. Lesson Procedure

### A. Common Mistakes Review (5–10 minutes)

- Use [Common-Mistakes-And-Debugging.md](../notes/Week-1/Common-Mistakes-And-Debugging.md) — **Session 4**.
- Fix 1–2 common errors: wrong path for `contact.html`, nav link to current page.

### B. Introduction of Core Concept (40–60 minutes)

Instructor explains:

- Why we have multiple pages (Home, Contact, etc.).
- Creating a second file (e.g. `contact.html`) with same structure.
- Copying header and nav; updating links (Home → `index.html`, Contact → `contact.html`).
- Relative paths: same folder vs subfolder.
- Live coding: add Contact page and link from Home.

### C. Guided Practice (30–45 minutes)

Students follow instructor to:

- Create `contact.html` with same `header` and `nav`.
- Set nav links: Home → `index.html`, Contact → `contact.html`.
- Add short contact text and a `mailto:` link on Contact page.

### D. Independent Practice (30–60 minutes)

**Main Weekly Exercise:**

Turn the Week 3 page into a 2-page site. Must include:

- **Home** (`index.html`): existing content.
- **Contact** (`contact.html`): same header and nav; short text and `mailto:` link.
- Same nav on both pages; all links work from either page.

**Real-world:** “Same idea as a small business or club site: Home + Contact.”

**Milestone:** Static Website Builder.

### E. Assessment

Students must demonstrate:

- Two pages with consistent nav.
- All links work from both pages.
- No broken paths.

### F. Wrap-Up Discussion (10–15 minutes)

- What was difficult?
- How do relative paths change when files are in different folders?
- How is this used in real life?

### G. Homework (Light / Optional)

- Skim Week 2 material on colors, fonts, and the box model. No required task.
