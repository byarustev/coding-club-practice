# 6-Week Web Development Bootcamp

Welcome to your journey into web development! This bootcamp is designed to take you from zero to building fully functional, interactive websites.

## 📚 Course Overview

This bootcamp covers the fundamentals of web development through hands-on learning:

- **Week 1**: HTML Fundamentals - Building the structure of web pages
- **Week 2**: CSS Fundamentals - Styling and making pages beautiful
- **Week 3**: Responsive Design & Flexbox - Making websites work on all devices
- **Week 4**: Introduction to JavaScript - Adding interactivity
- **Week 5**: DOM Manipulation & Events - Making pages dynamic
- **Week 6**: Final Project - Building a complete multi-page website

## 🎯 Learning Objectives

By the end of this bootcamp, you will be able to:

- Build semantic, well-structured HTML pages
- Style websites with CSS using modern best practices
- Create responsive layouts that work on mobile, tablet, and desktop
- Write JavaScript to add interactivity to web pages
- Manipulate the DOM to create dynamic user experiences
- Build complete, multi-page websites from scratch

## 📁 Course Structure

Each week includes:

1. **Teaching Materials** (`Week[X]-[Topic].md`) - Comprehensive lesson content
2. **Assignment Instructions** (`Week[X]-Assignment.md`) - Step-by-step project guide
3. **Code Examples** - Practical demonstrations throughout lessons

## 🛠️ Prerequisites

- A computer (Mac, Windows, or Linux)
- A text editor (VS Code recommended)
- A modern web browser (Chrome, Firefox, or Safari)
- Internet connection
- Enthusiasm to learn! 🚀

## 📖 How to Use This Course

1. **Read the weekly lesson** - Start with the teaching materials for each week
2. **Follow along with examples** - Type out code examples to build muscle memory
3. **Complete the assignment** - Apply what you've learned in a real project
4. **Review and iterate** - Don't be afraid to experiment and break things!

## 💡 Tips for Success

- **Practice daily** - Even 30 minutes a day makes a huge difference
- **Build projects** - The best way to learn is by doing
- **Ask questions** - There are no stupid questions
- **Read error messages** - They're trying to help you!
- **Use Google** - Professional developers Google constantly
- **Be patient** - Learning to code takes time and practice
